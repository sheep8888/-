# Classical convolutional neural networks
简单实现了LeNet5、AlexNet、Vgg、NiN、GoogLeNet、ResNet以及DenseNet，并将其应用到Fashion-Minst数据集
## 训练结果如下：
|  net   | train_acc  | test_acc  |
|  ----  | ----  |----  |
| LeNet  | 0.861 |  0.883 |
| AlexNet  | 0.864 | 0.862  |
| Vgg  | 0.939 | 0.918  |
| NiN  | 0.955 | 0.918  |
| GoogLeNet  | 0.963 | 0.921  |
| ResNet  | 0.990 | 0.916  |
| DenseNet  | 0.951 | 0.901  |
